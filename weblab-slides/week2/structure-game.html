Welcome to the HTML Structure Game!

Each section represents a main part of an HTML document, and your goal is to place the correct tags in their respective containers.

Drag elements from the list and drop them into the Head, Body, or Footer sections on the game board.

You'll receive instant feedback - a green message for correct placement and a red message for incorrect ones. Have fun learning HTML structure!
