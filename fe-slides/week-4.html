CSS Variables (Custom Properties)

Are User defined entities that contain values that can be reused througout the document.

  • Variables for the most part are defined in the :root element, and can be accessed anywhere in the document.
  • Variables are preceded by -- --primary-color{color:#efefef;}
  • Custom properties are case sensitive.
  • Use the variable by writing color: var(--primary-color);

Transitions and Animations

Transitions

Animations

Transitions

  • They go from A to B. No intermediate steps
  • Can only run once.
  • Require a trigger to run. No Autoplay.
  • The go back to the original state when the trigger is removed.
  • Easier to use with Javascript.
  • Transitions use the transition CSS property.

Animations

  • Go from A to B but can have steps in between.
  • Can repeat and loop indefinitely.
  • Can work with or without a trigger.
  • Can run in any direction. (Forwards, reverse, alternate).
  • Harder to use with Javascript.
  • Animations use the Animation CSS property and keyframes to determine the changes.

Transitions

  • .class {transition: property-name, duration, easing-function, delay;}
  • Easing functions: ease, ease-in, ease-out, ease-in-out, linear;

Animations

  • .class {animation: keyframe-duration, easing-function, delay, iteration count, direction, fill-mode, play-state, name;}
  • .class {animation: keyframe-name, easing-function, delay, iteration count, name;}

Reset for animation:

    
                        @media (prefers-reduced-motion: reduce){
                        *, *::before, *::after{
                        animation-duration: 0.01ms !important;
                        animation-iteration-count: 1 !important;
                        transition-duration: 0.01ms !important;
                        scroll-behavior: auto !important;
                        }

Libraries and Frameworks

Both are pre-written resusable code you can use in your projects.

Libraries

  • A library is a collection of functions or operations that solve a specific problem.
  • You can decide where to use a library in your project and which elements to use.
  • The whole estructure of your site does not depend on the library.
  • Examples of libraries are Animate.css, Font Awesome
  • Some CSS libraries.

Frameworks

  • A Framework is a structure in which you base your project.
  • You cannot decide where to use it. It is present all across your project.
  • Your projects depends on the Framework and you have a limited set op options.
  • Examples of Frameworks are: Bootstrap, Tailwind, Foundation.

Font Awesome

An icon library

  • Open a free account at Font Awesome
  • Create a kit
  • Paste the kit on the head document, just like you do with Google fonts.
  • You can use the icons in your text

Bootstrap

A CSS Framework that lets you create websites without writing CSS code.

  • It is responsive ready.
  • It resizes the images for you according to the viewport.
  • It has many ready-to-use components (slideshow, accordion, tab).
  • It works well with Javascript and JQuery.
  • It has extensive documentation.
  • It is customizable.

A no-brainer solution?

While using a CSS framework can be a way to speed up your workflow, there are a few things that you have to consider.

  • Using a CSS Framework is no replacement for learning CSS.
  • Bootstrap is very powerful, but the files can get big because it loads many things you might not need or use in your project.
  • It might not be the best solution for a custom site.
  • Sites can look generic.

How to use Bootstrap?

  • Go to the Bootstrap documentation page.
  • Copy the styles link on the head of your document
  • Copy the javascript CDN link before the closing body tag.
  • You can customize the basic CSS variables on your own css. BS CSS Variables.

Some Bootstrap rules

  • The container class gives you a pre-formatted area. They are required when using the grid.
  • BS uses the CSS Flex by default. Grid is optional in custom builds.
  • When using Flex, they have to be included in a row and the column class is col.
  • In Bootstrap left and right become start and end.
  • Use the class img-fluid to generate responsive images. When using picture to add multiple images, add the clas to the image tag.
