Libraries and Frameworks

Both are pre-written resusable code you can use in your projects.

Libraries

  • A library is a collection of functions or operations that solve a specific problem.
  • You can decide where to use a library in your project and which elements to use.
  • The whole estructure of your site does not depend on the library.
  • You can use libraries with frameworks.
  • Examples of javascript libraries are
    • JQuery
    • React.

Frameworks

  • A Framework is a structure in which you base your project.
  • You cannot decide where to use it. It is present all across your project.
  • Your projects depends on the Framework and you have a limited set op options.
  • Examples of javascript Frameworks are:
    • Angular JS
    • Ember JS
    • Vue JS

Should I use a library or a Framework?

  • Both libraries and frameworks require an understanding of Javascript.
  • Frameworks require a deeper understanding of Javascript.
  • Start with a library first. When you're comfortable with it you can move to a framework.

Jquery (Write less, do more.)

  • JQuery is a Javascript Library based on the JavaScript language.
  • It was the first widely used JavaScriot library.
  • Even though it's losing popularity, there are still many websites and applications that depend on JQuery.

Why do we use JQuery?

  • JQuery has been around since 2006 (JS was released in 1995).
  • It was created to solve a browser compatibility problem with JavaScript.
  • It made it a lot easier to interact with the DOM.
  • It let you add complex functionality to your website without having to write full javascript.
  • It's open source, so anyone can add to it.
  • It has an extensive plug-in library.

The jQuery library contains the following features:

  • HTML/DOM manipulation.
  • CSS manipulation.
  • HTML event methods.
  • Effects and animations.
  • AJAX (Asynchronous JavaScript And XML).
  • Utilities.
  • Lots of Plugins.

Parts of a JQuery script


<script>
$(document).ready(function(){
$(".myClass").html("This is a JQuery text");
});
</script>
  • <script> (Opening Tag)
  • $(document).ready(function(){ (Tells the script not to run until the document has finished loading).
  • document targets the current document.
  • $ to access JQuery.
  • (selector) Finds and selects the element in the HTML.
  • .action() the type of action we want to perform (Here we add what's being changed).

Basic JQuery Functions

.css();

Add a CSS property.

$(".box").css
("background-color", "blue");
.css({"color":"blue",
"font-size":"30px"});

Looks for the class .box on the html and adds a background color.

.addClass();

Add a CSS class.

$("#myId").addClass( "yourClass" );
$("#myId").addClass( "yourClass anotherClass" );

Looks for the id myId on the html and adds the css class.

.removeClass();

Remove a CSS class.

$("#myId").removeClass ("yourClass");
$("#myId").removeClass ("yourClass anotherClass");

Looks for the id myId on the html and removes the css class.

.html();

Replaces the HTML Content.

$("#myId").html
(" More information");

Looks for the id myId on the html and replaces the content.

Add Content

Adds content at the beginning or end of an element.

$("p").prepend("This will go at the beginning");
$("p").append("This will go at the end");

Adds content at the before or after an element.

$("p").before("This will go before");
$("p").before("This will go after");

Remove Content

Removes the element.

$(".element").remove();

Empties the content of the element without removing it.

$(".element").empty();

.width()

Alters the width of an element.

$("#myDiv").width("100px");

.height()

Alters the height of an element.

$("#myDiv").height("100px");

JQuery Events

.click()

Performs an action when clicked.

$( "#target" ).click(function() { $("p").hide(); });

Looks for the id on the html and add the interaction.

.dblclick()

Performs an action when double clicked.

$("#target").dblclick (function(){
$("p").hide(); });

.mouseenter()

Performs an action when The mouse enters the area of the selector.

$("#target").mouseenter (function(){ $("p").hide(); });

Looks for the id on the html and add the interaction.

.mouseleave()

Performs an action when The mouse enters the area of the selector.

$("#target").mouseleave(function()
{ $("p").show(); });

.hover()

Performs an action when The mouse hovers over the area of the selector.

$("#target").hover(function(){ $(this).css("color","red"); }, function(){ $(this).css("color","black"); }); });

The on() method

Allows you to attach more than one event handler to an element. Specially usefull when you need to restore the element to a previous state after the event is over.

$("p").on({ mouseenter: function(){
$(this).css("background-color", "lightgray");
},
mouseleave: function(){
$(this).css("background-color", "lightblue");
},
click: function(){
$(this).css("background-color", "yellow");
}
});

Basic Jquery Effects

.show() .hide()

Shows/Hides an element.

$("#clickBox").show();
$("#clickBox").hide();

Looks for the id myId on the html and shows or hides the element.

Slide

Slides an element Up and Down.

$( "#target" ).slideUp(speed, callback);
$( "#target" ).slideDown(speed, callback);
$( "#target" ).slideToggle(speed, callback);

Looks for the id myId on the html and shows or hides the element.

.fadeout() .fadein()

Fades an element.

$("#myDiv").fadeOut(1000);
$("#myDiv").fadeIn(1000);

Looks for the id myId on the html and fades it.

.toggle()

Toggles the action on the element.

$("#toggleBox").toggle(1000);

Looks for the id myId on the html and toggles the element.

.animate()

Allows you to create custom animations based on most css properties.

$("button").click(function(){
$("div").animate({left: '250px', opacity: '0.5'});
});

For positioning animation, the element needs to have a position attribute.
When animating 2 word properties (padding-left) remove the hyphen and use camelCase instead: paddingLeft.
