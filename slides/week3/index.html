Semantic Markup

HTML Text Elements that are not intended to affect the structure of your webpage, but add extra information to the page

Inline HTML Tags

  • Bold or Strong?

    Use <b> when you want to give <b>visual</b> contrast


    Use <strong> when you want to give <strong>importance</strong> to part of the paragraph

  • Italic or Emphasis?

    Use <i> when you want to give <i>visual</i> emphasis


    Use <em> when you want to give <em>emphasis</em> to a word

  • Span

    It is used to change the visual attributes of a group of words

    <p>This is a text <span>that will change color</span> in the same line</p>

    This is a text that will change colorin the same line

  • Cite

    When you need to reference a published work. It can be a book, movie, etc

    <cite>The David, by Michelangelo</cite>

    The David, by Michelangelo is in Florence, Italy

  • Subscript and Superscript

    Mostly used for formulas a math equations


    <sub> Water is H2O
    <sup> E = mc2

  • Inline quote

    Used to indicate quotations inside a paragraph.

    <p>Small quote inside a paragraph. He said:<q>It's cold in here</q></p>


    Small quote inside a paragraph. He said: It's cold in here

  • Quotations - blockquote

    Used to indicate quotations.

    <blockquote> Quam diu etiam furor iste tuus nos eludet?</blockquote>


    Quam diu etiam furor iste tuus nos eludet?

  • Figure and Figcaption

    Use it to group images and captions

    <figure>

    <img src="images/myImage.png" alt="add description here" width="50" height="50"/>

    <figcaption>This is the caption<figcaption>

    </figure>

  • Address

    Use <adress> when you want to give contact information
    <address>
    Your Name
    Your Address
    Phone Number
    Email
    </address>


The DIV tag

  • <div></div>
  • This is called the Division Tag
  • Is the most used tag in web development
  • It is a generic container tag used to separate and structure content
  • group of various tags of HTML so that sections can be created and style can be applied to them.
  • They are used with ids and classes for styling and positioning

Semantic structural tags

  • Article

    <article> </article>

    • • Represents a complete, or self-contained, composition in a page
    • • Can be independently distributed or reused
    • • Think of a magazine article

  • Section

    <section> </section>

    • Use it to group content

    • Can group articles

    • Can define parts of an article

    • Think of chapters on a book

  • Header

    <header> </header>

    • Marks the introductory content

    • Mostly used for the website top area

    • It can also mark the header of an article or section

  • Nav

    <nav> </nav>

    • Contains the navigation (menu) sections of your website.

    • Can be located anywhere in the page

  • Footer

    <footer> </footer>

    • Similar to headers, but at the end of a document

    • Mostly used for the website bottom area

    • It can also mark the footer of an article or section

  • Aside

    <aside> </aside>

    • Use it for extra information related to the content but not part of it

    • Mostly use for Side bars

    • Ads related to the content

ID's and Clases

These are CSS selectors that allow you to identify the HTML elements for styling and scripting

ID

  • Preceded by the # in the css
  • can only identify one element in the page
  • #blue can only be applied to one html tag
  • <p id="blue">This is the text with an id </p>

Class

  • Preceded by a . in the css
  • can identify multiple element in the page
  • .black can be applied to many html tags
  • <p class="black">This is a text with an class </p>
